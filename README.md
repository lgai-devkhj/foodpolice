# FoodPolice

## 🧩 프로젝트 소개
FoodPolice는 식품의 칼로리나 당류가 아닌 **원재료 기반 가공도(K-NOVA)**를 분석하여  
더 건강한 선택을 돕는 AI 서비스입니다.

---

## 🎯 문제 정의
현재 많은 사람들이 식품을 선택할 때  
칼로리, 당류 등 숫자 정보에만 의존하고 있습니다.

하지만 이러한 정보만으로는  
식품의 실제 가공 정도와 건강 영향을 충분히 판단하기 어렵습니다.

---

## 💡 해결 방법
이미지 기반 AI 분석을 통해  
식품의 원재료를 해석하고 가공도를 분류합니다.

- K-NOVA 기반 가공도 분석
- 초가공식품(4A, 4B, 4C) 세분화
- 주의 원재료 및 섭취 참고 제공

---

## ⚙️ 주요 기능
- 📸 **제품 분석**: 이미지 업로드 → 가공도 분석
- 🔍 **제품 비교**: 두 제품 가공도 비교
- 🔄 **대체 식품 추천**: 더 낮은 가공도 식품 제안
- 🧠 **퀴즈 기능**: 가공도 학습

---

## 🛠️ 기술 스택
- **Frontend**: TypeScript (Web App)
- **AI**: Gemini, Perplexity
- **Deployment**: Vercel

---

## 🚀 기대 효과
- 초가공식품 섭취에 대한 인식 개선
- 청소년의 건강한 식품 선택 지원
- 원재료 중심 식품 이해 확산
